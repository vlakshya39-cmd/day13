# Day 13 – AI-Powered Job Discovery with Claude & Indeed Connector

## Objective

Explore how AI can assist in job discovery, skill-gap analysis, and career planning using Claude AI and the Indeed Connector.

* Skills: C Programming, Prompt Engineering, GitHub

## Key Findings

* React and MERN Stack roles dominate the current internship market.
* AI/ML internships are available but usually require Python fundamentals.
* GitHub portfolios and project-based learning are highly valued by startups.

* Noida and Delhi NCR have strong demand for Web Development Interns.
